## 문제

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 입력

첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

## 출력

첫째 줄에 A+B를 출력한다.

## 예제 입력 1

```
1 2 
```

## 예제 출력 1

```
3
```

## 문제풀이

BufferedReader.readLine()을 사용하고 공백을 구분자로 A, B를 추출한다.

입력을 String 타입으로 받아주었기 때문에 계산을 위해 Integer.parseInt를 통해 int로 변환시켜주고 두 수의 합을 출력한다.
