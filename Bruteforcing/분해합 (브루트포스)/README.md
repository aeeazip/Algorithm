## 문제

어떤 자연수 N이 있을 때, 그 자연수 N의 분해합은 N과 N을 이루는 각 자리수의 합을 의미한다. 

어떤 자연수 M의 분해합이 N인 경우, M을 N의 생성자라 한다. 

예를 들어, 245의 분해합은 256(=245+2+4+5)이 된다. 

따라서 245는 256의 생성자가 된다. 

물론, 어떤 자연수의 경우에는 생성자가 없을 수도 있다. 

반대로, 생성자가 여러 개인 자연수도 있을 수 있다.

자연수 N이 주어졌을 때, N의 가장 작은 생성자를 구해내는 프로그램을 작성하시오.

## 입력

첫째 줄에 자연수 N(1 ≤ N ≤ 1,000,000)이 주어진다.

## 출력

첫째 줄에 답을 출력한다. 생성자가 없는 경우에는 0을 출력한다.

## 예제 입력 1

```
216
```

## 예제 출력 1

```
198
```

## 문제풀이

각 자리 수의 합이 최대일 때는, 모든 숫자가 9로만 구성된 경우이다. 

따라서 입력받은 수의 자릿수를 구하고 (입력받은 수 - 자릿수*9)부터 입력받은 수까지 반복문을 돌리며

해당하는 숫자가 입력받은 수의 생성자인지 검사하면 된다. 

입력받은 수의 생성자인지 검사하는 함수는 간단하다. 

매개변수로 들어오는 숫자에 각 자릿수에 해당하는 수를 더해주고 그 값이 입력받았던 수와 같은지 검사하면 된다. 

그런데 처음에 입력받은 수가 18보다 작은 경우는 (입력받은 수 - 자릿수*9)가 음수가 되기 때문에 검사를 시작할 수를 0으로 초기화 해주어야 한다.
